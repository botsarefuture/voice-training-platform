# Voice Training Platform for Transwomen

A comprehensive platform to help transwomen develop and practice their voice training, including pitch, tone, and speech rhythm adjustment.

## Features

- Audio recording and analysis
- Voice training modules
- Progress tracking
- Community features
- Offline capability (Whisper for local processing)

## Project Structure

```
voice-training-platform/
├── backend/          # Python backend with Flask
├── frontend/         # Web frontend interface
├── docs/             # Documentation
├── scripts/          # Utility scripts
└── README.md
```

## Installation

1. Install Whisper (local processing)
2. Set up Python virtual environment
3. Install dependencies
4. Run the application

## Development

This project is designed to be fully open source and community-driven.
